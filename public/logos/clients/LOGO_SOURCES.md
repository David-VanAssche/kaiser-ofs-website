## Instructions

1. Download SVG versions when available (better quality, smaller file size)
2. If only PNG available, use a tool like https://vectorizer.ai/ to convert
3. Optimize SVGs using https://jakearchibald.github.io/svgomg/
4. Ensure logos are single-color or work well in grayscale (component applies grayscale filter)
5. Recommended dimensions: ~200px wide, proportional height
